关于 HLOG

欢迎来到 HLOG！这是一个由 Cloudflare D1 驱动的现代化博客系统。

我们的初衷是打造一个极简、快速且功能齐全的内容分享平台。在这里，你可以记录技术心得、分享生活点滴，并与志同道合的朋友进行交流。

技术栈

  • 前端：纯原生 HTML + JavaScript (无框架负担)，搭配 Tailwind CSS 实现响应式设计。
  • 后端：基于 Cloudflare Workers，实现边缘计算极速响应。
  • 数据库：采用 Cloudflare D1 (Serverless SQLite)，零配置、高可用。
